以前用的vim字体字母和显示的样式都是粗体的，现在更改了一下vim的字体，中文字体的显示看的更加的舒服，英文的字母o0il显示的都非常的明显，非常的习惯这个字体。

我用了两个字体，用一个字体也行vim的配置如下

	set guifont=Droid_Sans_Mono:h13
	set guifontwide=YaHei_Mono:h12
